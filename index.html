Machine Learning in the Browser

About the Project

We at DSG decided to make various Machine Learning and Deep Learning Models interactive so that the user can visualise their inner working. Thus, Visual Machine Learning contains a set of Machine Learning and Deep Learning interactive visualisation demos for developing intuition. These demos are developed using Tensorflow.js and can be executed directly in your browser. We have made many new demos, as well as added various features on top of the exisiting tfjs examples.
